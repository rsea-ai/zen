#  Zen WordPress Theme

## 🎨 主题特色

Zen 一个基于极简主义设计理念构建的 WordPress 主题。它专注于排版、留白和沉浸式阅读体验，旨在为博主和读者提供一个无干扰的数字空间。

* 🏁 极简主义美学：基于 Tailwind CSS 构建，专注于排版、留白与字体的细腻处理，提供无干扰的沉浸式阅读体验。
* 🌓 响应式与深色模式：完美兼容移动端与桌面端，支持跟随系统自动切换 深色/浅色模式，随时随地保持舒适视觉。
* ♿️ 无障碍与 SEO 优化：内置 JSON-LD 结构化数据，严格遵循 A11y 标准（语义化标签、ARIA 支持、键盘导航优化），对搜索引擎和屏幕阅读器更友好。
* 💻 代码高亮体验：集成 Highlight.js，内置 GitHub Dark 配色，自动检测代码语言，为技术写作提供优雅的展示效果。
* 📚 丰富的独立页面：内置设计精美的「文章归档（时光机）」与「友情链接」独立模板，让内容组织井井有条。
* 📖 沉浸式阅读交互：配备顶部阅读进度条与右侧智能悬浮目录 (TOC)，支持滚动监听与平滑跳转，长文阅读游刃有余。
* 🎵 媒体样式增强：重构了 HTML5 音频播放器 UI，美化文件下载模块，并优化了 YouTube/Vimeo 等视频嵌入的响应式表现，细节处见真章。

## 🚀 安装主题

> Zen 主题最新压缩包请前往右侧 [Releases](https://github.com/rsea-ai/zen/releases) 下载。

1. 上传主题：在后台 外观 -> 主题 -> 安装主题，上传压缩包。
2. 启用主题：在后台 外观 -> 主题 中启用 Zen。
3. 配置菜单：
    * 在外观 -> 菜单 中创建一个菜单，显示位置勾选 Primary Menu。
    * 建议添加“首页”、“归档”、“友链”等页面。
4. 页面设置：
    * 新建页面，标题为“归档”，右侧模板选择 Archives Template。
    * 新建页面，标题为“友链”，右侧模板选择 Links Template。
5. 添加友链：
    * 主题已自动开启“链接 (Links)”管理器。在后台添加链接，设置名称、Web地址、描述和图像地址即可。
